# Example: Designing a Report from Scratch

## The Brief

Create a quarterly business report for stakeholders. Content includes:
- Executive summary (3 key findings)
- Revenue metrics (tables and charts)
- Customer metrics (tables and charts)
- Strategic initiatives (narrative)
- Next quarter outlook (narrative)

## Step 1: Information Structure

| Level | Content | Reason |
|-------|---------|--------|
| Primary | Executive summary with 3 key findings | Stakeholders scan first |
| Secondary | Revenue metrics | The core business data |
| Tertiary | Customer metrics | Supporting evidence |
| Quaternary | Strategic initiatives | Context for decisions |
| Quinary | Next quarter outlook | Forward-looking context |

**Key insight:** The executive summary must work as a standalone document. Many stakeholders will read only the first page.

## Step 2: Perceptual Design Decisions

### Hierarchy Decision

**Decision:** Three-tier heading system.
- Title: 24px, bold, dark color
- Section headings: 18px, bold
- Sub-headings: 14px, bold

**Perceptual rationale:** Pre-attentive processing. The size differences are large enough to be detected before reading. The reader can scan the page and understand the structure from layout alone.

### Grouping Decision

**Decision:** Each section is a distinct visual unit with consistent internal structure:
- Section heading (18px bold)
- 24px space
- Content
- 48px space before next section

**Perceptual rationale:** Gestalt proximity. The 24px internal spacing and 48px inter-section spacing create clear group boundaries. Related content is visually connected. Unrelated sections are visually separate.

### Rhythm Decision

**Decision:** Consistent spacing scale:
- 8px: between related items (list items, table rows)
- 16px: between paragraphs
- 24px: between heading and content
- 32px: between sub-sections
- 48px: between major sections
- 64px: after executive summary (signals transition to detail)

**Perceptual rationale:** Consistency and expectation. The reader can predict spacing patterns. Larger spacing signals more significant transitions.

### Emphasis Decision

**Decision:** Key findings in the executive summary get accent treatment:
- 16px, bold
- Accent color (blue)
- Left border (4px accent color)
- 16px padding on all sides

All other content uses neutral treatment.

**Perceptual rationale:** Pre-attentive processing + emphasis requires restraint. The accent color and border make findings pop. The restraint elsewhere makes the emphasis meaningful.

### Flow Decision

**Decision:** Single-column layout for narrative sections. Two-column layout for metrics (chart left, table right).

**Perceptual rationale:** Vertical flow for sequential content (narrative). Horizontal comparison for metrics (data that benefits from side-by-side comparison). The layout matches the reading pattern for each content type.

### Restraint Decision

**Decision:** No decorative elements. No icons unless they serve comprehension. Color limited to accent blue + neutrals. Font limited to one family (system font).

**Perceptual rationale:** Signal-to-noise. Every element earns its place. The restraint makes the emphasis on key findings more powerful.

|-----------|-------|-------|

## Step 4: Litmus Test Verification

Every decision is explainable:

| Decision | Perceptual Principle |
|----------|---------------------|
| 24px title | Pre-attentive processing — size is detected before reading |
| 48px between sections | Gestalt proximity — signals section boundaries |
| Accent color on findings | Pre-attentive processing — color pops out instantly |
| Single-column narrative | Continuity — natural vertical reading flow |
| No decorative elements | Signal-to-noise — nothing fights comprehension |
| Consistent section structure | Consistency — reader forms expectations |
